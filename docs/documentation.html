EduMaster - Documentation

Thank you for purchasing EduMaster. This documentation will help you set up and customize the template.

1. Introduction

EduMaster is a responsive landing page template designed for educational platforms, built using HTML, Tailwind CSS, and JavaScript.

2. Installation

Follow these steps to set up the template:

  1. Download and extract the files.
  2. Open index.html in your preferred web browser.
  3. Customize the design by editing css/style.css and js/script.js.

3. File Structure

  • index.html - Main landing page
  • css/style.css - Stylesheet
  • js/script.js - JavaScript functionality
  • images/ - Image assets
  • docs/ - Documentation files

4. Customization

Modify the styles in css/style.css and scripts in js/script.js as needed to fit your project.
