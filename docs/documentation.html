المميزات الرئيسية

  • تصميم متجاوب: يتكيف مع مختلف الأجهزة.
  • يدعم اللغة العربية (RTL): تصميم كامل للغة العربية.
  • بني باستخدام Tailwind CSS: سهل التخصيص.
  • أقسام منظمة: شريط تنقل، قسم بطل، مميزات، منهج دراسي، آراء العملاء، أسعار، مدرب، أسئلة شائعة، وتذييل.
  • أيقونات Font Awesome: لمسة جمالية واحترافية.
  • ملفات JavaScript منظمة: لتفعيل الخصائص الديناميكية.

التقنيات المستخدمة

  • HTML5
  • CSS3
  • Tailwind CSS
  • JavaScript
  • Font Awesome

التثبيت والاستخدام

  1. تنزيل القالب: أول خطوة تنزيل القالب بعد شراءه من موقع بيكاليكا القالب من [ حسابي على بيكاليكا ].
  2. فك الضغط: فك الضغط عن الملف الذي تم تنزيله.
  3. التعديل (اختياري): قم بتعديل ملفات HTML و CSS لتخصيص القالب.
  4. الرفع إلى الخادم: ارفع ملفات القالب إلى الخادم الخاص بك.

التخصيص

تعديل النصوص

يمكنك تعديل النصوص مباشرة في ملفات HTML. ابحث عن النص الذي تريد تعديله وقم بتغييره.

تغيير الألوان والخطوط

قم بتعديل ملف `css/style.css` لتغيير الألوان والخطوط. إذا كنت تستخدم Tailwind CSS بشكل متقدم، يمكنك أيضًا تعديل ملف `tailwind.config.js`.

إضافة صورك الخاصة

استبدل الصور الموجودة في مجلد `images/` بصورك الخاصة. تأكد من أن الصور بنفس الأبعاد للحفاظ على التصميم.

ملفات القالب

  • `index.html`: الملف الرئيسي الذي يحتوي على هيكل الصفحة.
  • `css/style.css`: ملف CSS الذي يحتوي على تنسيقات القالب الأساسية.
  • `js/script.js`: ملف JavaScript الذي يحتوي على التعليمات البرمجية الديناميكية.
  • `images/`: مجلد يحتوي على الصور المستخدمة في القالب.
  • `tailwind.config.js`: ملف تهيئة Tailwind CSS (إذا كان موجودًا).
